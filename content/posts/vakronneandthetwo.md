编者[^编者]语：在所有由我编纂的史前故事中，瓦克罗尼与沃拉斯/托恩尼双星的故事是最让我难以写出来的故事之一。并不是因为这段历史的记录过于稀少，也不是由于这段历史被扭曲了很多，正相反，战争几方的记录都相对属实，且相对完备，至少能大致了解到曾经的情况[^完备的历史资料]，但这段历史实在太过沉重，我很难以平静的心情去叙述。

<br/>

### 费泽与韦德的星系之争

费泽与韦德两颗行星均位于沃拉斯/托恩尼双星系，费泽星是双星系统的第二颗行星，而韦德是第四颗，相对距离双星更遥远，也跟寒冷。现已有证据表明费泽人和韦德人大概率出于同源，但并不是所有人都赞同这个观点，并且是哪颗星球上先出现了类似的生命，以及当时的人是怎么移居去另一个星球的，这些都不得而知[^费泽与韦德人的源起]。

而目前我们能考据到的即是双方在隐藏纪元中便已开始了战争，但总体规模都不算大。史学家称其为“沃拉斯/托恩尼双星内战”。沃拉斯/托恩尼双星内战最开始的原因和导火索均已不可查：有一些费泽的传说中写是天象的指引；也有一些表示在那个时期附近沃拉斯/托恩尼双星系统进行过一次叠加耀斑，影响了费泽的生态环境，迫使费泽人与韦德人争夺生存资源；有一些韦德遗录表示长期以来从费泽进口的粮食和农产品贸易价格高涨，引起了韦德人的不满；还有一些指明两行星之间在争夺双星系统有限的星际空间资源……

也很有可能，他们谁都不清楚到底是什么先发生了，什么后发生了，但总之是发生了很多事情，两星之间稀里糊涂地便开战了。在隐藏纪元后期和战争纪元初期，沃拉斯/托恩尼双星内战愈演愈烈，战局从一开始的二者持平，演变成了韦德星逐渐占了上风。从历史遗物上可以推知局面倾斜的一大原因便是费泽星的自然环境愈加恶劣，农作物均生出锈斑，河水干枯，海啸频繁，火山剧烈，费泽人民不聊生，生活疾苦，再加上长时间的拉锯战，损失和补给不对等，费泽星陷入了劣势的状态。韦德人曾向费泽提出和谈，但要求中包含了一系列政治和经济贸易上的条件[^韦德的要求]。而费泽人认为，已经进行了上百年的双星内战不能就如此屈辱而又一无所获地结束，于是费泽拒绝了和谈，选择了与韦德星背水一战。

但不出所料，在战争纪元中前期，费泽的力量被彻底击垮，剩余的费泽太空船全部用于搭载着部分费泽人逃往其他星域。

沃拉斯/托恩尼双星内战之后，韦德人一直统治着两个星球。但这个时候费泽星的状态已是低谷，有很多费泽人死在了天灾人祸中，在这期间韦德人也为费泽的生态环境修复做出了努力，但效果并没有立竿见影。同时，费泽人依旧憎恨韦德星，这个时期有很多费泽地下组织兴起，但由于资源和经济生活水平的限制，这些地下组织并没有做出能改写历史的举动。

而那批踏上逃亡之路的费泽人则承载着所有费泽人的希望，他们永远都不会忘记那耻辱的一天，他们心中装着复仇的怒火，想要某一天带着强盛的人民把费泽星夺回来。但实际上他们中的任何一个再也没有回到他们曾经热爱的家园。

韦德人也保持着对费泽人的仇恨，他们瞧不起与他们同星系的友邻，而对于部分费泽人乘船出逃的行为更是愤怒。在内战结束之后韦德人一刻也没有放松，他们继续搜寻着费泽飞船的痕迹，企图找出他们的方向，因他们惧怕外来的援兵，并且他们还不想很快就失掉这次的胜利。

<br/>

### 瓦克罗尼

而在瓦克罗尼的记载中，费泽人先是被他们成为Fhazo（瓦克罗尼语中费泽星的音译，费左），后是被称为“法砍的费左”（Fhazo el Faccam，法砍即灾星之意）。

瓦克罗尼是一个非常富饶的星球，它的富饶与强大一直延续至今，可以说放眼整个宇宙的可知文明，瓦克罗尼无论在经济、军事还是科技、思想上都一直处于领先宇宙平均水平的状态。而在战争纪元中期，逃亡的费泽人出于机缘巧合被瓦克罗尼的舰队拦下，后来彼此都没有敌意，瓦克罗尼便将这群费泽人带到了他们的本土星球。瓦克罗尼人认为他们是难民，并且据说费泽人的飞船也已经航行了一二百年之久，船体破烂不堪，船载补给也即将见底。瓦克罗尼人把北半球的一块并不大的陆地空出来给费泽人居住，他们对这些“难民”的经历和文化十分感兴趣。在费泽人到来之前，瓦克罗尼没有遇到过任何一个外星种族，这主要是由于瓦克罗尼的星域并不拥挤，其中的可居住行星也相对较少，但他们有自己的太空舰队和统一的编制，这是因为瓦克罗尼人一直在尝试寻找地外生命的存在，并且预想到了对方可能并不友好，他们对自己的恒星系的搜索和监视非常严密，这也是为什么他们能第一时间发现费泽的船队。

费泽人带来了一些跨星系航行的技术和知识，还传授给瓦克罗尼自己母星的文化，包括语言。大多数瓦克罗尼人都非常乐于接受，但也有少数瓦克罗尼文献中提到，这是一件“迷人又危险的馈赠”，并且“魅惑了瓦克罗尼好奇求知的心”[^迷人又危险的馈赠]。

那曾是瓦克罗尼在星联纪元前最后的欢乐时期，有的史学家称之为“加墨埃多内” （Gamo'eDonne[^Gamo'eDonne]，意为告别童年），但也有很多史学家声称瓦克罗尼的童年在他们开始建造太空舰队的时候就已经结束了。

后来在战争纪元309年，韦德人追踪着费泽的航迹侵入了瓦克罗尼所在的柯维诺星系，对瓦克罗尼人宣战。瓦克罗尼此前从未与其他星球交战过，以至于他们一开始非常慌张，自乱阵脚。同时大多数瓦克罗尼人都认为这是法砍的费左带来的灾难，于是在同年，瓦克罗尼便杀死了所有借住在瓦克罗尼的费泽人，并被迫面对韦德的侵略。

瓦克罗尼在自己的恒星系共有四道哨关，由外至内分别位于纳沃洛星（Navolo）、欧迈尔星（Omael）、塔勒斯星（Talleth）和瓦克罗尼的唯一的卫星埃赛维（Esseve）上，这四个哨关也是柯维诺星系最主要的军事基地。欧迈尔星与塔勒斯星轨道中间有一段小行星带。纳沃络星和欧迈尔星均为气态巨行星，而柯维诺星系其余四颗行星均为固态。总体来说瓦克罗尼的位置属于易守难攻。

但是由于韦德星军队的到来令没有经验的瓦克罗尼措手不及，于是韦德星在短时间内就攻占了纳沃络星和欧迈尔星的哨关，但是在次年韦德星准备一鼓作气出兵攻占塔勒斯哨关的时候，他们受到了小行星带的阻碍，韦德星所在星系是一个双星恒星系，附近没有小行星带的存在，而这天然的星系结构让瓦克罗尼扳回一局，使战局暂时稳定了下来。

战争纪元322年，瓦克罗尼重整旗鼓，发誓要夺回自己星系的领地。他们向韦德的舰队发动了进攻，虽然他们仅有30舰，要对战韦德星的一百余舰，但这些年间，瓦克罗尼星发生了一次“技术爆炸”，他们的武器和舰船等科技均有了巨大的进步，但是在这么短的时间内他们只能利用有限的资源去完成自己的复仇。

出乎意料的是，这次“欧迈尔哨关之战”中瓦克罗尼取得了来之不易的胜利，这是整个宇宙可知文明中著名的以少胜多之战。韦德见状全军撤退出柯维诺星系，回到了沃拉斯/托恩尼双星系。

战争纪元386年，瓦克罗尼舰队前往沃拉斯/托恩尼双星系，完成自己的复仇。他们称沃拉斯/托恩尼双星系为“格里姆尼”（GliMune[^Glimune]，意为诅咒之源）。在瓦克罗尼进攻沃拉斯/托恩尼双星系的时候，韦德的军队已经衰落，在此之前韦德刚经历了一场内乱，反对战争的党派与政府交战，在这次内乱中韦德失去了非常多的军事力量，甚至比对外战争的时候的损失还要惨重。韦德舰队仅剩的两个连16艘星舰和普通的运输船抵御瓦克罗尼的精锐部队长达三个月的时间，但他们最终还是失败了。这场悲壮的交锋被韦德历史称作“洛梅的十六簇火”，又或是“洛梅的十六颗星”[^洛梅]。

在韦德失败之后，瓦克罗尼对韦德星和费泽星进行了屠杀和破坏，文明的历史尽数被毁，仅有不到一半的人躲过了瓦克罗尼的残杀。

<br/>

### 韦德和费泽的复仇

在瓦克罗尼的屠杀之后，韦德和费泽站在了统一战线，二者合作重建了家园，保存了文明的火种。两星球放下了彼此之间的恩怨，均将矛头指向了瓦克罗尼星，并渴望着报仇。

在战争纪元435年，韦德和费泽出军征战柯维诺星系，史称“迈尔科洛斯”（Milcoloth[^Milcoloth]，意为正义讨伐之征）。“迈尔科洛斯”持续了整整63年，在这场战争中无论是韦德、费泽还是瓦克罗尼都元气大伤，长时间的战争本就令双方感到疲惫，加之主战场距离沃拉斯/托恩尼双星系并不近，韦德和费泽一方便愈加艰难。

在“迈尔科洛斯”中，有一首韦德的歌谣流传至今，仍让韦德人回想起那悲壮的历史：

> ……迈洛[^Milo]的意志，普摩[^Pmogh]的力量
<br/>
今日吾等复荣光
<br/>
瓦克荣[^vakron]，瓦瑟乔[^vathechol]
<br/>
秋收的吉光照耀双日殿堂[^双日殿堂]
<br/>
前人的灵魂保佑吾等
<br/>
韦德之歌必将传唱……

在这场僵持了63年的战争中双方两败俱伤，最终韦德和费泽的军队由于后援和补给被断，主部队失利。最终在塔勒斯哨关处瓦克罗尼一举歼灭了所有敌军，自此之后塔勒斯哨关又被称作“最后战场”（即“卡罗塔利埃”，CaloTaliee[^Calotaliee]）。

而韦德星和费泽星因这场长久的失败的战争一蹶不振，加上瓦克罗尼所要求的赔偿条款和残暴的罪名，沃拉斯/托恩尼双星系此后的历史都像蒙上了一层厚重的阴影，直到加入星际联盟并接受了“建设者”计划之后情况才有所好转。

[^编者]: 《史前故事集》由因希尔的历史学家拉斐西尔（Laphecil）写成，于星联纪元278年首次出版，后来在星联纪元311年重新整理后再版。这里刊登的所有故事集文章皆来源于视界·地平线出版社的第九版。据说拉斐西尔在外游历几十年，亲自调查这些失落的历史和传说，才写下这部书。————译者注，以下没有特殊说明即为原作者注。
[^完备的历史资料]: 这大概是由于瓦克罗尼和费泽、韦德人平均寿命均高于可知宇宙文明的平均水平。瓦克罗尼人的寿命大概在600年到800年之间不等，而韦德人寿命在500年上下，费泽人在300年到500年之间。
[^费泽与韦德人的源起]: 对这方面感兴趣的读者可以自行查阅《沃拉斯/托恩尼双星之谜》这本书，里面包含了有关此现象的约20种不同的猜想，此书的作者之一谢菲（Syephay）仍在此领域的前沿进行研究。
[^韦德的要求]: 韦德人的这些和谈的条件并没有任何记录能将其留在历史中，现今的说法多是传说与传闻，其中有多少被扭曲也不得而知。
[^迷人又危险的馈赠]: 这两句话皆出自《法砍的费左的诡计》一书，作者为那哈尔多·墨瑞那，首次出版于战争纪元277年。
[^Gamo'eDonne]: 瓦克罗尼语。gamon，动词，告别，再见，名词形式即为它的词根Gamo。Donne，意为童年，少年。“e”表示“对……，对于……”。
[^Glimune]: Glim，意为诅咒，谶语。Une，源头，始于。
[^洛梅]: 洛梅是韦德神话中的一位被神偏爱的人类使者。在当时一名叫做德茨莫特的神为了惩罚世人对祂的忽视，为世界带来了极寒，同时也夺走了世界的光明。洛梅请求神祗原谅世人，他攀登了最高的雪山，来到众神之殿门前跪坐。三年后非常喜爱他的光芒之神埃塞利松心软了，于是打开大门答应了洛梅的请求。洛梅带着火光回到人类之中，这些火光缓缓上升，成为了韦德夜空中的星星，光芒吸引了斯卡洛和斯卡莫（韦德神话中的太阳神，双胞胎），于是祂们再次普照了韦德大地。后来洛梅通常被韦德人看作是希望的象征。
[^Milcoloth]: Mil，韦德语，意为正义，正当。colo，意为讨伐，惩罚，通常具有褒对贬的含义，coloth，colo的名词。
[^Milo]: 迈洛，Milo，韦德神话中的正义之神，正义Milane的词根Mil来源于此。
[^Pmogh]: 普摩，Pmogh，韦德神话中的复仇之神。
[^vakron]: 瓦克荣，Vakron，瓦克罗尼的韦德语说法。
[^vathechol]: 瓦瑟乔，Vathechol，韦德语，意为黑暗拥护者。Vath，即黑暗。echo，拥护，拥趸，尾缀-l表示“拥护……的”。
[^双日殿堂]: 双日殿堂，即沃拉斯/托恩尼双星系外环的韦德-费泽太空站，是双星历史上规模最大的太空基地，由对称的两个半圆柱体组成，由于它的宏伟和光辉让人们想到神话中的太阳神双子斯卡洛和斯卡莫，故得此名。
[^Calotaliee]: calo，瓦克罗尼语，最终的。tali，战斗，战争。Taliee，战场，沙场。
